# Fieldline

CRM SaaS for trade businesses (lawn care, HVAC, plumbing) in NW Ohio / SE Michigan.

## Stack
- Next.js 14 (App Router)
- Upstash Redis
- Anthropic Claude (claude-sonnet-4)
- Iron Session
- Tailwind CSS

## Features (v0.5)
- Customers with lifetime value tracking
- Leads with kanban + drag/drop + AI scoring
- AI Daily Briefing
- AI Quote / Follow-up / Price Suggestion
- Calendar with recurring jobs
- Map view (OpenStreetMap)
- Analytics (revenue, funnel, sources, projections)
- Invoices with public customer-facing links
- Quote acceptance via public link
- CSV import/export
- Bulk lead actions
- Activity timeline per lead/customer/global
- Global search (⌘K)
- Mobile-responsive
